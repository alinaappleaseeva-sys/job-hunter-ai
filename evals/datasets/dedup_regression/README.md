# dedup_regression (Phase 5/8)

Must-merge and must-not-merge pairs for dedup quality, now extended for cross-family (ATS + boards + Telegram).

## Format
- must_merge: list of pairs (or groups) that should collapse to same canonical.
- must_not_merge: pairs that must stay separate.
- cross_family: examples mixing ATS + board + repost.
