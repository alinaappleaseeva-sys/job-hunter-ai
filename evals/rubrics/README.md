# Evaluation Rubrics

Rubrics define what counts as success or failure.

Examples:

- what counts as a duplicate
- what counts as a ghost job (see ghosting_precision.md)
- what counts as a relevant job for a profile (see ranking_topk.md)
- what counts as an extraction error versus acceptable fallback
